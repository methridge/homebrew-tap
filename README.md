# Methridge Tap

## How do I install these formulae?

`brew install methridge/tap/<formula>`

Or `brew tap methridge/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "methridge/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
